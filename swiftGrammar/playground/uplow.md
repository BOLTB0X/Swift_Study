## lowercased()

> Returns a lowercase version of the string.
> <br/>

소문자들을 대문자로 변환해주 반환
<br/>

```swift
let tmp = "BBQ"
print(tmp.lowercased()) // bbq

// 시간복잡도 O(n)
```

## uppercased()

> Returns an uppercase version of the string.
> <br/>

```swift
let tmp2 = "bbq"
print(tmp2.uppercased()) // bbq

// 시간복잡도 O(n)
```
