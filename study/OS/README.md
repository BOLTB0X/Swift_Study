# OS 특집

**OS 관련 핵심 개념들 정리**

> **동시성** : 1명이 여러 일을 빠르게 번갈아 처리 (요리, 전화, 청소를 왔다갔다) <br/>
**병렬성** : 여러 명이 각자 일을 동시에 함 (요리 담당, 전화 담당, 청소 담당) <br/>
**멀티프로세스** : 여러 독립된 사람(프로세스)이 각자 집에서 일함 -> 독립적, 통신은 느림 <br/>
**멀티스레드** : 같은 집(프로세스) 안의 여러 사람이 함께 일함 -> 공유는 빠르지만 충돌 위험 있음

| 개념                           | 비유                                            | 설명                                                               |
| ---------------------------- | --------------------------------------------- | ---------------------------------------------------------------- |
| **멀티태스킹 (Multitasking)**     | “한식, 일식, 중식 등 여러 요리를 동시에 준비하는 주방”             | 전체 주방(=CPU)이 여러 종류의 요리를 동시에 처리하는 것처럼 보이게 함. 실제로는 번갈아가며(시분할) 조리함. |
| **멀티프로세싱 (Multiprocessing)** | “여러 명의 요리사가 각자 주방을 맡아서 요리하는 상황”               | 주방이 여러 개(CPU/Core가 여러 개)라서 **진짜 동시에 병렬 실행** 가능.                  |
| **멀티프로세스 (Multi-process)**   | “한식 주방, 일식 주방이 각각 독립된 공간에서 요리하는 것”            | 각 주방(프로세스)은 **독립된 재료·도구(메모리)** 를 씀. 한쪽이 망가져도 다른 쪽 영향 X.          |
| **멀티스레드 (Multi-thread)**     | “한식 주방 안에서 밥 담당, 반찬 담당, 국 담당이 동시에 일하는 것”      | 같은 주방(프로세스) 내에서 재료(메모리)를 공유하며 협업. 빠르지만, 칼이나 불을 같이 써서 충돌 위험 있음.   |
| **동시성 (Concurrency)**        | “요리사가 한 가지 재료 손질하다가 불 끄고 다른 재료 손질하고 또 돌아오는 식” | 실제로 동시에는 아니지만, **빠른 전환으로 동시에 하는 것처럼 보임**                         |
| **병렬성 (Parallelism)**        | “여러 요리사가 동시에 다른 재료를 손질”                       | **물리적으로 동시에 실행되는 상태** (진짜 동시 실행)                                 |

### in iOS

| 구분                | 설명                                                                                                    |
| ----------------- | ----------------------------------------------------------------------------------------------------- |
| **프로세스(Process)** | iOS에서는 **앱 하나 = 하나의 프로세스**<br>앱을 실행하면 iOS가 독립된 메모리 공간을 할당하고, 그 안에서 코드가 실행 됌                   |
| **스레드(Thread)**   | 프로세스 안에서 실제 코드를 실행하는 흐름 단위<br>앱 실행 시 **기본적으로 메인 스레드(Main Thread)** 가 하나 생성되어 UI, 이벤트, 런루프 등을 처리  |
| **메인 스레드의 특징**    | UI 업데이트, 터치 이벤트, UIKit 접근은 **무조건 메인 스레드에서만 가능**.<br>시간이 오래 걸리는 작업을 메인 스레드에서 실행하면 -> UI 멈춤(“앱이 멈춘 것처럼”) |
