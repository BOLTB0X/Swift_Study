## DTO(Data Transter Object)

- data 전송/교환을 위한 객체
  <br/>
- data를 Object로 변환하는 객체
  <br/>
- A.swift->B.swift, AView->BView처럼 전달하는 작업을 처리하는 Object
  <br/>

- method 호출 횟수를 줄이기 위해 데이터를 담고 전달하는 것, 가변성인 특징이 있음

## VO(Value Object)

- 값을 위해 사용하는 Object
  <br/>

- 읽기를 하기 위해서 사용하는 객체
  <br/>

- 간단한 독립체를 의미하는 작은 객체를 의미
  <br/>

- 값이 같으면 동일한 Object라고 볼 수 있음
  <br/>

- 불변성
